# testimonials

Audio clips and avatar photos for Section 3 (the testimonials carousel) live here.

The placeholder data in `lib/testimonials.ts` references two audio clips that are
not committed yet:

- `/testimonials/audio-1.mp3`
- `/testimonials/audio-2.mp3`

Until those files exist, the inline audio player degrades to a disabled
"audio coming soon" state — nothing errors. Drop the real `.mp3` files here (and
any avatar images you point `avatar:` at) to light them up.
